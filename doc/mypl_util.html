log/5

log(Module::atom(), Line::integer(), X3::Level, Msg::string(), Params) -> integer()

  • Level = debug | dbg | normal | inf | warn | wrn | error | err
  • Params = [term()]

Add information in the log streams. If Level is dbg, print message on stdout; else use the standard application error_logger (levels stands for info, warning and error).
This function is used through applications macros (adviserl.hrl) which automaticaly capture Level, Module and Line.
